# Frontend

Based on [esbuild](https://esbuild.github.io/), [TypeScript](https://www.typescriptlang.org/), [React](https://react.dev/) and [MUI](https://mui.com/).

Requirements.
* `Node v22.11.0`
* `npm v10.9.1`

Configuration.
```
frontend/public/config.json
```

Install.
```
cd frontend
npm install
```

Run.
```
cd frontend
npm run start
```

Build.
```
cd frontend
npm run build
```

Clean.
```
npm run clean
```
