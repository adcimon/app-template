# Backend

Based on [TypeScript](https://www.typescriptlang.org/), [Node.js](https://nodejs.org/en/) and [NestJS](https://nestjs.com/).

Requirements.
* `Node v22.12.0`
* `npm v11.0.0`

Configuration.
```
backend/.env
```

Install.
```
cd backend
npm install -g @nestjs/cli
npm install
```

Run.
```
cd backend
npm run start
npm run start:dev
npm run start:debug
npm run start:prod
```

Build.
```
cd backend
npm run build
```
