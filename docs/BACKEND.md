# Backend

Based on [TypeScript](https://www.typescriptlang.org/), [Node](https://nodejs.org/en/) and [Nest](https://nestjs.com/) with [AWS](https://aws.amazon.com/).

Configuration.
```
backend/.env
```

Install.
```
cd backend
npm install -g @nestjs/cli
npm install
```

Start.
```
cd backend
npm run start
npm run start:dev
npm run start:debug
npm run start:prod
```

Build.
```
cd backend
npm run build
```
