# Frontend

Based on [TypeScript](https://www.typescriptlang.org/), [React](https://react.dev/) and [MUI](https://mui.com/).

Configuration.
```
frontend/public/config.json
```

Install.
```
cd frontend
npm install
```

Run.
```
cd frontend
npm start
```

Build.
```
cd frontend
npm run build:prod
```
